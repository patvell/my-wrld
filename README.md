# my-wrld 

Built using Google Antigravity.

## 🚀 Tech Stack
- **Framework:** Next.js 16 (Turbopack)
- **Database:** Local SQLite (using better-sqlite3)
- **Styling:** Tailwind CSS (if applicable)

## 🛠 Setup instructions
1. **Clone the repo:** `git clone https://github.com/patvell/my-wrld.git`
2. **Install dependencies:** `npm install`
3. **Database:** The local SQLite database (`my-wrld.db`) will be automatically created in the root directory on the first API request.
