# my-wrld 

Built using Google Antigravity.

## 🚀 Tech Stack
- **Framework:** Next.js 16 (Turbopack)
- **Database:** Supabase
- **Styling:** Tailwind CSS (if applicable)

## 🛠 Setup instructions
1. **Clone the repo:** `git clone https://github.com/patvell/my-wrld.git`
2. **Install dependencies:** `npm install`
3. **Environment Variables:** Create a `.env.local` file and add your Supabase keys:
   ```text
   NEXT_PUBLIC_SUPABASE_URL=your_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
